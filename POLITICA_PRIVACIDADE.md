# Política de Privacidade - Beach Tennis Super 8

**Última atualização:** Dezembro 2024

---

## 1. Introdução

O **Beach Tennis Super 8** ("nós", "nosso", "aplicativo") respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações quando você usa nosso aplicativo.

## 2. Informações que Coletamos

### 2.1 Dados Coletados pelo Aplicativo

O Beach Tennis Super 8 coleta e armazena as seguintes informações **apenas localmente no seu dispositivo**:

- **Nome de jogadores:** Nomes dos participantes dos eventos que você cria
- **Informações de eventos:** Nome, data, local e configurações dos eventos
- **Resultados de jogos:** Placares e resultados das partidas
- **Rankings:** Classificações e estatísticas dos participantes

### 2.2 Dados NÃO Coletados

O aplicativo **NÃO coleta**:
- Informações de localização (GPS)
- Dados de contato (telefone, e-mail)
- Informações de pagamento
- Dados de navegação ou uso
- Informações de dispositivos
- Identificadores únicos de dispositivo

## 3. Como Usamos Suas Informações

### 3.1 Uso dos Dados

Os dados coletados são usados **exclusivamente** para:
- Permitir que você gerencie eventos e torneios de Beach Tennis
- Calcular rankings e estatísticas dos participantes
- Gerar jogos e organizar rodadas
- Armazenar resultados e histórico de eventos

### 3.2 Armazenamento Local

**Todos os dados são armazenados localmente no seu dispositivo** usando SQLite. Nenhuma informação é enviada para servidores externos ou compartilhada com terceiros.

## 4. Compartilhamento de Dados

### 4.1 Sem Compartilhamento

O Beach Tennis Super 8 **não compartilha, vende ou transfere** seus dados pessoais para terceiros. Todos os dados permanecem exclusivamente no seu dispositivo.

### 4.2 Funcionalidade de Compartilhamento

O aplicativo oferece funcionalidade de compartilhamento de rankings e resultados. Esta funcionalidade usa o sistema nativo de compartilhamento do seu dispositivo e **você tem controle total** sobre o que compartilhar e com quem compartilhar.

## 5. Segurança dos Dados

### 5.1 Proteção Local

Como todos os dados são armazenados localmente no seu dispositivo, a segurança depende das medidas de segurança do próprio dispositivo:
- Use senha/PIN/biometria no seu dispositivo
- Mantenha o sistema operacional atualizado
- Use antivírus se necessário

### 5.2 Sem Transmissão de Dados

Como não há transmissão de dados para servidores externos, não há risco de interceptação de dados durante transmissão.

## 6. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem os seguintes direitos:

### 6.1 Acesso aos Dados

Você pode acessar todos os seus dados a qualquer momento através do aplicativo. Todos os dados estão armazenados localmente e você tem acesso completo.

### 6.2 Exclusão de Dados

Você pode excluir seus dados a qualquer momento:
- **Excluir evento:** Remove todos os dados relacionados ao evento
- **Excluir participante:** Remove o participante e seus dados
- **Desinstalar aplicativo:** Remove todos os dados do dispositivo

### 6.3 Portabilidade dos Dados

Atualmente, o aplicativo não oferece exportação automática de dados. Futuras versões podem incluir funcionalidade de exportação.

### 6.4 Revogação de Consentimento

Como não coletamos dados pessoais sensíveis e não compartilhamos dados, não há necessidade de consentimento explícito. Você pode parar de usar o aplicativo a qualquer momento e excluir todos os dados.

## 7. Permissões do Aplicativo

### 7.1 Permissões Utilizadas

O Beach Tennis Super 8 **não requer permissões especiais** do sistema Android. O aplicativo funciona sem acesso a:
- Localização
- Câmera
- Microfone
- Contatos
- Arquivos externos
- Internet

### 7.2 Funcionamento Offline

O aplicativo foi projetado para funcionar **completamente offline**. Não é necessária conexão com a internet para usar o aplicativo.

## 8. Dados de Crianças

O Beach Tennis Super 8 não coleta intencionalmente dados de crianças menores de 13 anos. Como todos os dados são armazenados localmente e não há coleta online, não há risco de coleta inadvertida de dados de crianças.

## 9. Alterações nesta Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Quando fizermos alterações significativas, notificaremos você através de:
- Atualização do aplicativo
- Nota nas notas de versão na Play Store

A data da "Última atualização" no topo desta política indica quando a política foi revisada pela última vez.
